# 🔮 贱命算命师

一个长按算命的搞笑网页游戏

## 游戏特色

- 🎯 长按水晶球3秒进行算命
- 📱 完美适配手机端
- 😏 8种贱命一条的搞笑结果
- ✨ 炫酷的视觉效果和动画

## 如何游玩

1. 长按水晶球
2. 等待进度条满载
3. 查看你的"贱命"结果
4. 点击"再来一卦"继续算命
